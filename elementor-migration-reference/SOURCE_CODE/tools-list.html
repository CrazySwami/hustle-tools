generate_json_patch
Modification
Generates RFC 6902 JSON Patch operations to apply specific changes to the Elementor JSON. This is used for ALL JSON modifications - it creates surgical, line-level edits instead of rewriting the entire file.
When AI Uses It:
"Change the heading color to red"
"Make the button text say 'Get Started'"
"Add a new widget to the layout"
"Remove the text editor widget"
Parameters:
{ "patches": [ { "op": "replace", "path": "/content/0/settings/title_color", "value": "#ff0000" } ], "summary": "Change heading color to red" }

analyze_json_structure
Query
Analyzes the current Elementor JSON structure and provides information about widgets, properties, and locations. Used for answering questions without modifying anything.
When AI Uses It:
"What widgets do I have?"
"Where is the button color defined?"
"How many heading elements are there?"
"Find all widgets with red colors"
Parameters:
{ "query_type": "find_property", "target": "title_color" } Types: - "find_property" - "list_widgets" - "get_widget_info" - "search_value"

open_template_in_playground
Action
Opens or refreshes the Elementor template in WordPress Playground. Automatically switches to the Playground tab, launches WordPress if needed, and shows your template live.
When AI Uses It:
"Show me the template"
"Open this in playground"
"Preview these changes"
"Refresh the preview"
Parameters:
{ "action": "launch", "message": "Opening preview..." } Actions: - "launch" - Start playground - "refresh" - Update template - "open_editor" - Open Elementor
